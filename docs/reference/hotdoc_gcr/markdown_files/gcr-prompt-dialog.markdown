### GcrPromptDialog

a GTK+ dialog prompt

 A [](GcrPrompt) implementation which shows a GTK+ dialog. The dialog will
 remain visible (but insensitive) between prompts. If the user cancels the
 dialog between prompts, then the dialog will be hidden.
